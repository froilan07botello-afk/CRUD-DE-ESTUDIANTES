

## Descripcion del proyecto

El proyecto consistió en practicar un crud para un sistema de estudiantes el cual requeria dar de alta,baja y consultas a estudiantes, al igual que sus carreras. Como primer paso se crearon las migraciones necesarias para poder hacer funcionar nuestra bd, para despues pasar a configurarlas y crear sus modelos , vistas y controladores, despues fuimos implementando las rutas necesarias para cada una de las vistas que tendria nuestro sistema para poder comenzar a crear las funcionalidades dentro de nuestro controlador que manejara todos los datos y funcionalidades especificas tales como validaciones,consultas etc.

A su vez creamos y organizamos las vistas necesarias para mostrar datos y funcionalidades que se piden en la descripcion del problema.
